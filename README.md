ğŸ“ˆ Description of the Data

The U.S. Bureau of Labor Statistics (BLS) collects national employment and unemployment data dating back to the early 1940s.
This particular dataset lists the employment status of the civilian noninstitutional population aged 16 and over from 1953 to 2023.
It includes variables such as:

Year

Total population

Number employed

Number unemployed

Unemployment rate (percentage)

This dataset allows us to extract, compare, and visualize unemployment percentages across different years.
